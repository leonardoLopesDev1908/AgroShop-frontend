# AgroShop – Descrição do Front-end

O front-end do AgroShop foi desenvolvido com foco em experiência do usuário, desempenho e organização visual, oferecendo uma navegação fluida e clara para clientes e administradores. A interface é totalmente responsiva e utiliza práticas modernas de desenvolvimento, garantindo compatibilidade com dispositivos desktop e mobile.

A aplicação foi construída em React, usando componentes reutilizáveis, estados bem organizados e consumo eficiente da API. A comunicação com o backend é feita por meio de requisições REST seguras e padronizadas, garantindo que os dados exibidos estejam sempre atualizados.

O usuário tem acesso a várias funcionalidades importantes, como:

Página de produtos com listagem dinâmica e filtros práticos.

Detalhes completos de cada produto, exibindo descrição, preço, imagens, disponibilidade e avaliações.

Sistema de avaliações, onde clientes podem registrar título, comentário e nota em estrelas.

Exibição de comentários e notas de outros usuários, trazendo transparência e confiabilidade à plataforma.

Carrinho de compras com gerenciamento intuitivo de itens.

Processo de checkout simples e objetivo.

Login e registro de usuários, com tratamento visual de erros e feedbacks claros.

Dashboard administrativo, permitindo visualizar vendas, produtos mais vendidos e avaliações.

Além disso, o front-end segue boas práticas de design, com estrutura limpa, uso eficiente de React Hooks, tratamento centralizado de erros, customização de estilos via CSS e integração fluida com componentes de UI como cards, tabelas e modais.
O resultado é uma interface amigável, moderna e eficiente, ideal para um e-commerce profissional.
